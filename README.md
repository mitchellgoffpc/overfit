# Underfit Frontend

The frontend for [Underfit](https://github.com/underfit-ai/), an open-source model reporting dashboard for tracking experiments, metrics, and artifacts. Built with React, TypeScript, Vite, and Tailwind CSS.

## Getting Started

```bash
npm install
npm start
```

The dev server proxies API requests to `localhost:8000`, so make sure the [backend](https://github.com/underfit-ai/api) is running.

## Scripts

- `npm start` - Start the development server
- `npm run build` - Build for production
- `npm run preview` - Preview the production build
- `npm test` - Run tests
- `npm run typecheck` - Type-check with TypeScript
- `npm run lint` - Lint with ESLint
